# Retail Sales Analysis — SQL Analytics

Business analytics on the classic **Northwind** retail dataset. The focus is analytical
judgment — turning a business question into the *right* metric — not SQL syntax.

This repository is available in two languages. Each folder is a complete, self-contained
version (queries, reasoning, notebook, charts).
